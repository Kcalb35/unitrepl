# V1

## 输入

- `unitrepl` 直接进入交互
- `unitrepl "<expr>"` 直接输出结果

## REPL交互
* `help`：展示支持的命令以及输入样例
* `list`：展示所有支持的**基本单位符号**（按量纲分组：长度、时间）
* `exit`：退出 REPL
* `<expr>`: 按换算指令语法解析并执行
  - 解析或换算失败时输出错误信息，并继续下一次输入（REPL 不退出）。


## expr 语法
`<number> <from_unit> to <to_unit>`
  * `10 km to m`

### 数字
支持常见的小数、指数

### 单位
* 支持**复合单位**：必须显式使用 `*` 或 `/`
  * 例如：`m/s`、`km*h`（仅示例）
* 支持单位的指数（可为负），使用 `^`，暂时只支持整数次幂
  * 例如：`cm^-1`、`s^2`
* **不支持隐式乘法/空格乘法**：
  * 复合单位之间必须用 `*` 或 `/`
  * 不允许通过空格表示乘法
